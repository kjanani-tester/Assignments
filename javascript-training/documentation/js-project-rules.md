1.Project Folder Naming Convention (Root Folder)

Convention: kebab-case (lowercase + hyphen)
ex: javascript-training-concepts

2.Sub Folder /Module Naming Convention
Convention: kebab-case (lowercase + hyphen) or Camel Case
src/
page-objects
pageMethods (camel case - starts with lowercase followed by uppercase char before each new word)


3.JS/TS File Naming Convention
Convention : kebab-case (lowercase + hyphen)
Ex: first-js-program.js
