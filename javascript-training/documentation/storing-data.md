# What is Program?
A program is all about set of instructions that we are going to share with our computer to complete a specific task.

Ex:
1.Launch the chrome browser
2.Enter the URL " www.google.com "
Enter the keyword "Playwright" in the "Google search" text box.
Press Enter button and capture the test results.

# What is Instruction ?
Instruction is a combination of data plus respective action.

Example:
1.Launch the "Chrome" browser

data: Chrome
action : Launch
